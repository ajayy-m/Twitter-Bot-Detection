System Architecture
Classification Pipeline
1
Raw Feature Ingestion
30+ profile & behaviour signals loaded from API dataset: follower counts, tweet metrics, engagement ratios, profile completeness flags.
2
Feature Engineering
Derived signals computed: tweets/day, followers/day, spike scores, log-transforms, young account & popularity flags, activity scores.
3
Scaling & Normalisation
RobustScaler applied to tabular features; MinMax normalisation on account age. Handles outliers gracefully without distortion.
4
Graph Embedding Lookup
k-NN (k=10, cosine) finds similar training accounts; their 64-d node embeddings are averaged and L2-normalised to form a graph context vector.
5
Random Forest Classifier
Full feature vector passed to trained Random Forest.
6
LIME Explainability
Local surrogate model (300 perturbation samples) identifies the 15 features most responsible for this specific prediction.
7
LLM Narrative
Top LIME evidence translated into a plain-English analyst summary by a language model via OpenRouter API.

Interpreting LIME

LIME perturbs input features and measures prediction change. Bars extending right (green) push toward HUMAN; bars extending left (red) push toward BOT. Bar length shows relative influence — the longer the bar, the stronger that signal's effect on this specific prediction.
